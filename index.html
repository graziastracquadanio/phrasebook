The Phrasebook app is a web-based application that enables users to access a custom phrasebook, containing phrases they have created.

User

This section contains all of the APIs for managing users.

Creates a new user

This API is used to create a new user.

Request Body schema: application/json
required
username
string

The user supplied username

email
string <email>

The user email address

password
string

The user's plain text password.

Responses

https://example.com/v1/api/signup

Request samples

  • Payload
Content type
application/json
{
  • "username": "string",
  • "email": "user@example.com",
  • "password": "string"
}

Response samples

  • 201
  • 400
  • 500
Content type
application/json
{
  • "user": {
    • "username": "string",
    • "email": "user@example.com",
    • "id": "string"
    }
}

Phrase

This section contains all of the APIs for managing phrases.

Create a new phrase

Adds a new phrase to the phrasebook.

Request Body schema: application/json
text
string

The text of the phrase.

translation
string

The translation of the phrase.

notes
string

Additional notes attached to the phrase.

ownerId
string

The id of the user who created the phrase.

Responses

https://example.com/v1/api/phrases

Request samples

  • Payload
Content type
application/json
{

  • "translation": "string",
  • "notes": "string",
  • "ownerId": "string"
}
